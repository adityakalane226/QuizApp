# QuizApp 

QuizApp is an interactive web-based quiz game built using **HTML, CSS, and JavaScript**.  
Users can answer multiple questions and get instant feedback on their score at the end.

# Overview
This project provides a simple quiz experience in the browser.  
Questions are displayed one at a time, and users can select their answer from multiple choices.  
At the end of the quiz, the total score is shown — making it a great beginner project for practicing **DOM manipulation**, **event handling**, and **JavaScript logic**.

##  Features
-  Multiple-choice quiz questions  
-  Interactive answer selection  
-  Displays final score at the end  
-  Clean and responsive UI  
-  Works completely in the browser (no backend)

## Tech Stack
- **HTML5** – structure and layout  
- **CSS3** – design and styling  
- **JavaScript** – game logic and interaction
